# EuroSAT 遥感图像分类项目 (HW1)

本项目使用多层感知机 (MLP) 对 EuroSAT 数据集进行土地覆盖分类。

## 1. 项目实现步骤

### 数据处理 (`dataset.py`)
- **数据集加载**: 自定义 `EuroSATDataset` 类，使用 `PIL` 加载图像。
- **预处理**: 将图像缩放至 64x64，转换为 Tensor 并进行归一化处理（均值 0.5，标准差 0.5）。
- **数据增强**: 训练集加入了随机水平翻转 (RandomHorizontalFlip)、随机垂直翻转 (RandomVerticalFlip) 和随机旋转 (RandomRotation)，以提高模型的泛化能力。
- **数据划分**: 按照 8:1:1 的比例将数据集划分为训练集、验证集和测试集。

### 模型架构 (`model.py`)
- **增强型 MLP 模型**:
    - **层数加深**: 使用了 `[1024, 512, 256]` 三层隐藏层。
    - **Batch Normalization**: 在每个线性层后添加了 BN 层，加速收敛并稳定训练过程。
    - **Dropout**: 添加了 Dropout 层 (rate=0.3) 以防止过拟合。
- **输入层**: 将 64x64x3 的图像展平为 12288 维向量。
- **输出层**: 10 个类别。

### 训练与评估 (`train.py`)
- **损失函数**: 使用交叉熵损失函数 (CrossEntropyLoss)。
- **优化器**: 使用带动量的随机梯度下降 (SGD)，学习率为 0.01，动量为 0.9。
- **正则化**: 使用 L2 权重衰减 (Weight Decay = 1e-4) 防止过拟合。
- **训练轮数**: 增加至 20 个 Epoch。

## 2. 运行结果

- **测试集准确率**: 约 68.78% (20 个 Epoch 的深度训练结果)。
- **可视化结果**:
    - `learning_curves.png`: 展示了 Loss 和 Accuracy 随 Epoch 的变化曲线。
    - **混淆矩阵**: 展示了模型在 10 个类别上的分类混淆矩阵。

### 权重可视化与空间模式 (`visualize_weights.py`)
- **操作**: 将第一层隐藏层的权重矩阵（形状为 1024x12288）中的每一行提取出来，恢复成 3x64x64 的图像尺寸，并进行归一化显示。
- **观察与讨论**:
    - **色彩倾向**: 可视化结果显示，某些神经元的权重表现出明显的色彩偏好。例如，偏绿色的模式可能在提取“森林 (Forest)”或“草地 (HerbaceousVegetation)”的特征；偏蓝色的模式可能与“河流 (River)”或“海湖 (SeaLake)”相关。
    - **空间特征**: 部分权重图呈现出点状或条纹状的纹理，这表明模型正在学习识别地物的边缘、纹理（如农田的条纹）或特定的空间分布模式。
    - **局限性**: 由于 MLP 缺乏卷积层的空间局部性，这些权重图看起来比较杂乱，不如 CNN 的卷积核那样直观，但仍能反映出模型对颜色和粗略空间频率的敏感性。

## 3. 错误分析 (Error Analysis)

通过运行 `error_analysis_visual.py`，我们提取并可视化了测试集中分类错误的典型样本（见 `error_examples/misclassified_samples.png`）。

### 典型错例讨论：
1.  **Highway (高速公路) vs River (河流)**:
    *   **原因**: 两者在遥感图像中都呈现为狭长的线性结构。如果河流的颜色较深或反光，其视觉特征与柏油路面非常相似。由于 MLP 只是将像素展开为向量，缺乏捕捉局部拓扑结构（如河流的弯曲度 vs 公路的直线度）的能力，因此极易混淆。
2.  **HerbaceousVegetation (草地) vs Residential (住宅区)**:
    *   **原因**: 许多“住宅区”样本中包含大量的绿化带或草坪。当建筑物较小或分布稀疏时，全局颜色特征以绿色为主，导致模型误判为草地。
3.  **PermanentCrop (永久性作物) vs AnnualCrop (年度作物)**:
    *   **原因**: 这两类地物在光谱特征（颜色）上高度重合，主要的区别在于纹理的细微差异（如果园的行列间隔）。MLP 很难在不使用卷积核的情况下提取这种精细的纹理特征。
4.  **SeaLake (海湖) vs River (河流)**:
    *   **原因**: 两者本质都是水体。当河流较宽或湖泊较窄时，它们的像素组成几乎完全一致，模型难以仅凭单幅 64x64 的图像区分其地理形态。

### 总结：
MLP 的主要局限性在于它丢失了图像的空间结构信息。对于地物分类任务，颜色虽然是重要特征，但在面对形状相似或背景复杂的样本时，MLP 的表现远不如卷积神经网络 (CNN)。
